Per-Ankh

Analytics for Old World

"House of Life" in ancient Egyptian

Download

Features

Game Analytics

Track your progress across multiple games with detailed statistics and historical data.

Interactive Charts

Visualize victory points, resource production, and military strength over time.

Nation Comparisons

Compare performance across different nations and civilizations.

Save File Import

Simply import your completed Old World save files to start analyzing.

How It Works

  1. Find your saves - Locate your Old World save files
  2. Import - Select a folder containing completed games
  3. Explore - View charts, statistics, and historical data

Built With

Per-Ankh is a native desktop application built with Tauri, Rust, and Svelte for fast, efficient performance.
