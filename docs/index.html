Installation

Mac

  1. Download the .dmg file from Releases
  2. Open the .dmg and drag Per-Ankh to your Applications folder
  3. Launch from Applications

Windows

  1. Download the .msi file from Releases
  2. Run the installer
  3. If Windows Defender SmartScreen appears:
    • Click "More info"
    • Click "Run anyway"

    This warning appears because the app isn't code-signed. We are actively working on this issue.

Linux

Debian/Ubuntu:

  1. Download the .deb file from Releases
  2. Install with: sudo apt install ./per-ankh_*.deb

Fedora/RHEL:

  1. Download the .rpm file from Releases
  2. Install with: sudo dnf install ./per-ankh-*.rpm

Built With

Per-Ankh is a native desktop application built with Tauri, Rust, and Svelte for fast, efficient performance.
